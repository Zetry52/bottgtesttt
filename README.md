# Telegram Video Downloader Bot

Полноценный Telegram-бот на Python с использованием `aiogram 3.x`, который скачивает видео из TikTok, YouTube и Instagram через `yt-dlp`.

## Возможности

- Команда `/start` с приветствием, логотипом и кнопкой запуска.
- Команда `/help` с краткой инструкцией.
- Только inline-кнопки под сообщениями.
- FSM для хранения выбранной платформы.
- Отдельная логика скачивания для TikTok, YouTube и Instagram.
- Валидация ссылок до скачивания.
- Понятные ошибки при неверной ссылке, блокировке контента и сбоях.
- Сообщение о загрузке перед отправкой видео.
- Проверка лимита размера файла для Telegram.
- Временные файлы с автоматическим удалением после отправки.
- Логирование в консоль и в файл `logs/bot.log`.

## Установка зависимостей

1. Убедитесь, что у вас установлен Python 3.11+.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Как вставить токен

1. Создайте файл `.env` рядом с `main.py`.
2. Скопируйте в него содержимое из `.env.example`.
3. Вставьте ваш Telegram Bot Token:

```env
BOT_TOKEN=ваш_токен_бота
```

Важно: токен читается только из переменной окружения `BOT_TOKEN` и не хранится в коде.

## Запуск бота

```bash
python main.py
```

## Как работает бот

1. Пользователь нажимает `📥 Скачать видео`.
2. Выбирает платформу: TikTok, YouTube или Instagram.
3. Бот просит отправить ссылку.
4. Бот скачивает видео через `yt-dlp`.
5. Если файл проходит по размеру, бот отправляет его в чат.
6. После отправки временные файлы удаляются автоматически.

## Примечания

- Для некоторых видео YouTube доступный формат может зависеть от ограничений платформы.
- Если ролик приватный, удалённый, заблокирован или недоступен для скачивания, бот покажет понятное сообщение.
- Если размер видео превышает лимит отправки Telegram, бот предупредит об этом отдельно.
